CREATE TABLE IF NOT EXISTS entities (
    id         INTEGER PRIMARY KEY,

    name       TEXT NOT NULL,
    username   TEXT,
    updated_at INTEGER NOT NULL   -- Unix timestamp (int(time.time()))

WAL mode pitfall (RESEARCH.md §Pitfall 6): use `isolation_level=None` on connect, then execute `PRAGMA journal_mode=WAL`, then switch back. Pattern:
```python
conn = sqlite3.connect(db_path, isolation_level=None)
conn.execute("PRAGMA journal_mode=WAL")
conn.isolation_level = ""   # back to transactional
```
TTL pitfall (RESEARCH.md §Pitfall 3): store and compare Unix integer timestamps only (`int(time.time())`). No datetime objects in TTL comparison.

TTL values (REQUIREMENTS.md CACH-01): 30 days for users (2_592_000 seconds), 7 days for groups/channels (604_800 seconds).

Cursor spec (RESEARCH.md §Pattern 3):
```python
import base64, json

def encode_cursor(message_id: int, dialog_id: int) -> str:
    payload = json.dumps({"id": message_id, "dialog_id": dialog_id})
    return base64.urlsafe_b64encode(payload.encode()).decode()

def decode_cursor(token: str, expected_dialog_id: int) -> int:
    data = json.loads(base64.urlsafe_b64decode(token.encode()))
    if data["dialog_id"] != expected_dialog_id:
        msg = f"Cursor belongs to dialog {data['dialog_id']}, not {expected_dialog_id}"
        raise ValueError(msg)
    return data["id"]
```
