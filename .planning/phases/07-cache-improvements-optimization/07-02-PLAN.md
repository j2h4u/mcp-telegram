Purpose: ListMessages calls GetMessageReactionsListRequest to fetch reactor names; when the same messages are queried again within 10 minutes, cached names can be used instead of a fresh API call (saves latency and API quota).
Output: ReactionMetadataCache class in cache.py, table schema in entity_cache.db, integration point in tools.py ListMessages handler.
