Formatter spec from RESEARCH.md ¬ßPattern 4:
- Accepts `messages: list` (Telethon Message objects, newest-first from Telethon)
- Accepts `reply_map: dict[int, object]` ‚Äî message_id ‚Üí Message for reply annotation
- Accepts `tz: ZoneInfo | None = None` ‚Äî defaults to UTC
- Reverses messages internally (`for msg in reversed(messages)`) ‚Äî oldest-first for display
- Session break threshold: 60 minutes (`SESSION_BREAK_MINUTES = 60`)
- Date header format: `--- YYYY-MM-DD ---`
- Session break format: `--- N –º–∏–Ω ---` (N = integer minutes)
- Message line: `HH:mm SenderFirstName: text`
